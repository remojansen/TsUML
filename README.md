# TsUML

:construction: WORK IN PROGRESS :construction:

Generate UML diagram for your TypeScript applications powered by https://yuml.me/

![](/assets/cli-preview.gif)

## Installation

```sh
npm install -g tsuml
```

## Usage

```
tsuml --glob ./src/**/*.ts
```
